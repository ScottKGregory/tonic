## What is Tonic?
Tonic is a set of helpers and pre-built endpoints helpful for setting up a Gin based website/API.

Primary stack is:
  - [Gin](https://github.com/gin-gonic/gin)
  - [Zerolog](https://github.com/rs/zerolog)
